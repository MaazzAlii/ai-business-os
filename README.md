# AI Business OS

AI-powered Business Operating System designed for small businesses.

## Features

- Invoice Automation
- AI Business Assistant
- RAG Knowledge Base
- Supplier Management
- Inventory Tracking
- Customer Support Agent
- Daily Business Reports
- Business Analytics

## Tech Stack

- n8n
- FastAPI
- PostgreSQL
- ChromaDB
- Mistral AI
- Streamlit
- Docker
